# 如何在專案根目錄開啟終端機

這份指南會詳細說明如何在您的專案資料夾中開啟終端機（Terminal 或 PowerShell）。

## 📍 什麼是「專案根目錄」？

專案根目錄就是包含以下檔案的那個資料夾：
- `package.json`
- `README.md`
- `app/` 資料夾
- `lib/` 資料夾
- `.gitignore`

在您的電腦上，路徑應該是：
```
C:\Users\tina0\OneDrive\桌面\20251214_WEB
```

---

## 🪟 Windows 方法（3 種方式）

### 方法 1：使用檔案總管（最簡單）⭐

1. **開啟檔案總管**（按 `Win + E` 或點擊資料夾圖示）

2. **找到您的專案資料夾**
   ```
   C:\Users\tina0\OneDrive\桌面\20251214_WEB
   ```

3. **在資料夾內空白處，按住 `Shift` 鍵 + 右鍵點擊**

4. **選擇「在此處開啟 PowerShell 視窗」或「在此處開啟終端機視窗」**

   ![Windows 右鍵選單示意圖]
   - 如果看到「在此處開啟 PowerShell 視窗」→ 點擊它
   - 如果看到「在此處開啟終端機視窗」→ 點擊它

5. **會開啟一個黑色視窗，這就是終端機！**

6. **確認路徑**：視窗中應該顯示類似這樣的路徑：
   ```
   PS C:\Users\tina0\OneDrive\桌面\20251214_WEB>
   ```

---

### 方法 2：使用檔案總管位址列

1. **開啟檔案總管，進入專案資料夾**

2. **點擊位址列**（顯示路徑的地方）

3. **輸入 `powershell` 或 `cmd`，然後按 Enter**

4. **終端機會自動在該資料夾開啟**

---

### 方法 3：使用開始功能表

1. **按 `Win + X` 或右鍵點擊開始按鈕**

2. **選擇「Windows PowerShell」或「終端機」**

3. **在終端機中輸入以下命令切換到專案目錄**：
   ```powershell
   cd "C:\Users\tina0\OneDrive\桌面\20251214_WEB"
   ```

4. **按 Enter**

5. **確認路徑**：輸入 `pwd` 或 `cd` 查看當前路徑

---

## 🍎 Mac 方法（2 種方式）

### 方法 1：使用 Finder（最簡單）⭐

1. **開啟 Finder**

2. **找到您的專案資料夾**

3. **在資料夾內空白處，按住 `Control` 鍵 + 右鍵點擊**（或雙指點擊觸控板）

4. **選擇「新增位於資料夾位置的終端機視窗」**

5. **終端機會自動開啟，並且已經在正確的目錄**

---

### 方法 2：使用終端機應用程式

1. **開啟「終端機」應用程式**
   - 按 `Cmd + Space` 搜尋 "Terminal"
   - 或從「應用程式」→「工具程式」→「終端機」

2. **在終端機中輸入以下命令**：
   ```bash
   cd ~/Desktop/20251214_WEB
   ```
   或如果您的路徑不同：
   ```bash
   cd "/Users/您的用戶名/OneDrive/桌面/20251214_WEB"
   ```

3. **按 Enter**

4. **確認路徑**：輸入 `pwd` 查看當前路徑

---

## ✅ 如何確認您在正確的目錄？

在終端機中輸入以下命令來確認：

### Windows (PowerShell)
```powershell
# 查看當前路徑
pwd

# 或
Get-Location

# 查看資料夾內容
ls
# 或
dir
```

### Mac/Linux
```bash
# 查看當前路徑
pwd

# 查看資料夾內容
ls
```

**您應該看到以下檔案/資料夾：**
- `package.json`
- `README.md`
- `app/`
- `lib/`
- `.gitignore`

如果看到這些檔案，就表示您在正確的位置！✅

## 💡 小技巧

### Windows 快捷鍵
- 在檔案總管中，在位址列輸入 `cmd` 或 `powershell` 然後按 Enter
- 在資料夾中，按住 Shift + 右鍵，選擇「在此處開啟 PowerShell」

### Mac 快捷鍵
- 在 Finder 中，按 `Cmd + Option + C` 複製路徑
- 在終端機中，輸入 `cd ` 然後按 `Cmd + V` 貼上路徑

---

## 🆘 遇到問題？

### 問題 1：找不到「在此處開啟 PowerShell」選項
**解決方法：**
- 確保您按住 `Shift` 鍵再右鍵
- 或使用方法 2（在位址列輸入 `powershell`）

### 問題 2：終端機顯示「找不到路徑」
**解決方法：**
- 檢查路徑是否正確
- 確認資料夾名稱是否有空格（需要用引號包起來）
- 例如：`cd "C:\Users\tina0\OneDrive\桌面\20251214_WEB"`

### 問題 3：不知道專案資料夾在哪裡
**解決方法：**
1. 開啟 VS Code 或您的編輯器
2. 查看左側檔案列表最上方的資料夾名稱
3. 那就是您的專案根目錄

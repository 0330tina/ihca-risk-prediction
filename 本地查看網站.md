# 💻 如何在本地查看網站

這份指南會告訴您如何在您的電腦上預覽網站。

---

## 📋 前置準備

### 1. 確認已安裝 Node.js

在終端機執行：

```bash
node --version
```

**應該會顯示版本號**（例如：`v20.0.0`）

**如果沒有安裝：**
- 前往 [https://nodejs.org](https://nodejs.org)
- 下載並安裝 LTS 版本
- 重新開啟終端機

---

### 2. 確認已安裝依賴

在專案根目錄（`20251214_WEB`）執行：

```bash
npm install
```

**這會安裝所有需要的套件**（只需要執行一次）

---

## 🚀 啟動開發伺服器

### 步驟 1：開啟終端機

在專案根目錄開啟終端機：
- 在檔案總管中，進入 `20251214_WEB` 資料夾
- 按住 `Shift` + 右鍵
- 選擇「在此處開啟 PowerShell 視窗」

---

### 步驟 2：啟動開發伺服器

在終端機執行：

```bash
npm run dev
```

**預期輸出：**
```
  ▲ Next.js 14.x.x
  - Local:        http://localhost:3000
  - Network:      http://192.168.x.x:3000

  ✓ Ready in X seconds
```

---

### 步驟 3：開啟瀏覽器

1. **開啟瀏覽器**（Chrome、Edge、Firefox 等）

2. **前往以下網址：**
   ```
   http://localhost:3000
   ```

3. **您應該能看到網站了！** 🎉


## 🆘 常見問題

### Q: 顯示「port 3000 is already in use」？

**A:** 有其他程式在使用 3000 埠口。

**解決方法：**
1. 關閉其他使用 3000 埠口的程式
2. 或使用其他埠口：
   ```bash
   npm run dev -- -p 3001
   ```
   然後前往 `http://localhost:3001`

---

### Q: 顯示「command not found: npm」？

**A:** Node.js 沒有正確安裝。

**解決方法：**
1. 安裝 Node.js：https://nodejs.org
2. 重新開啟終端機
3. 再次執行 `npm run dev`

---

### Q: 顯示「Cannot find module」錯誤？

**A:** 依賴沒有安裝。

**解決方法：**
```bash
npm install
```

然後再次執行：
```bash
npm run dev
```

---

### Q: 網站顯示空白或錯誤？

**A:** 檢查：
1. 終端機是否有錯誤訊息
2. 確認所有檔案都已儲存
3. 嘗試重新啟動開發伺服器

## 💡 小技巧

### 在手機上查看

1. **找到您的電腦 IP 位址**
   - Windows: 在終端機執行 `ipconfig`，找到 IPv4 位址
   - Mac: 在終端機執行 `ifconfig`，找到 inet 位址

2. **確保手機和電腦在同一個 Wi-Fi 網路**

3. **在手機瀏覽器輸入：**
   ```
   http://您的IP位址:3000
   ```
   例如：`http://192.168.1.100:3000`

---

# 🔄 開發工作流程說明

## ✅ 正確的工作流程

### 第一次啟動

1. **執行一次**：
   ```bash
   npm run dev
   ```

2. **保持終端機開啟**
   - 開發伺服器會持續運行
   - 不要關閉終端機視窗

3. **開啟瀏覽器**
   - 前往 `http://localhost:3000`
   - 保持瀏覽器開啟

---

## 🔥 熱重載（Hot Reload）

### 修改程式碼後

**不需要重新執行 `npm run dev`！**

1. **修改檔案**（例如：`app/input/page.tsx`）
2. **儲存檔案**（`Ctrl + S`）
3. **瀏覽器自動更新** ✨

**就是這麼簡單！**

## 🛑 何時需要重新啟動？

### 需要重新執行 `npm run dev` 的情況：

1. **關閉了終端機視窗**
   - 開發伺服器停止了
   - 需要重新啟動

2. **修改了 `package.json`**
   - 添加或移除依賴
   - 需要重新安裝：`npm install`
   - 然後重新啟動：`npm run dev`

3. **修改了 `next.config.js`**
   - Next.js 配置檔案
   - 需要重新啟動

4. **修改了環境變數檔案**（`.env`）
   - 需要重新啟動

5. **遇到錯誤無法自動恢復**
   - 有時需要手動重啟

---

## ❌ 不需要重新啟動的情況

### 這些修改會自動更新：

- ✅ 修改 `.tsx`、`.ts` 檔案（React 元件）
- ✅ 修改 `.css` 檔案（樣式）
- ✅ 修改 API 路由（`route.ts`）
- ✅ 修改任何程式碼檔案

**儲存後立即生效！**

---

## 💡 實用技巧

### 技巧 1：多個終端機視窗

如果需要在開發伺服器運行時執行其他命令：

1. **開啟新的終端機視窗**
2. **在同一個資料夾中執行其他命令**
3. **原來的開發伺服器繼續運行**

---

### 技巧 2：查看終端機輸出

開發伺服器會顯示：
- ✅ 編譯狀態
- ✅ 錯誤訊息
- ✅ 警告訊息
- ✅ 頁面載入時間

**注意終端機的訊息，可以幫助您除錯！**

---

### 技巧 3：快速重啟

如果需要快速重啟：

1. **在終端機按 `Ctrl + C`**（停止伺服器）
2. **按上方向鍵**（顯示上一個命令）
3. **按 Enter**（重新執行 `npm run dev`）

## 🆘 常見問題

### Q: 修改後瀏覽器沒有更新？

**A:** 檢查：
1. 終端機是否還在運行？
2. 是否有錯誤訊息？
3. 嘗試手動重新整理瀏覽器（`F5` 或 `Ctrl + R`）

### Q: 終端機顯示錯誤？

**A:** 
1. 查看錯誤訊息
2. 修正錯誤
3. 儲存檔案
4. 通常會自動恢復

### Q: 想停止開發伺服器？

**A:** 
- 在終端機按 `Ctrl + C`
- 輸入 `Y` 或直接按 `Enter`
